Hexo 部署到 Github 文件夹名大小引起 404 问题

原因: 由于 git 命令默认的配置中忽略了文件名的大小写，因此即便文件夹的大小写发生了变更，git 也会表现出置之不理。

解决办法: 修改配置, 使 git 不要忽略大小写.
